Shapes

Bag

Bags have a “filled” aspect, round, they clearly contains other waste (garbage bags, shopping bags…)
Crumpled plastic bags, partially visible, partially visible, should be in crumpled foil - please see examples in the corresponding section.

Examples


Crumpled foil

The emphasis is on crumpled. Any crumpled foil or sheet, as well as torn plastic or foil (but not the torn compostable, which has its own label).
✅ Crumpled plastic bags are crumpled foil as well!
❌ Foil or sheet that are not crumpled go in the "sheet" category.

Examples


Sheet

Flat objects with sharp edges, semi-rigid aspect, plane, no volume
Includes for example sheets of paper (as long as not crumpled) or flat pieces of plastic.
❌ Flat disks go in "lid / cap / disk"
❌ Crumpled foil (even crumpled paper) go in "crumpled foil"

Examples


Box

3D boxes and cartons. Includes post boxes, Tetra Pak, trasnport crates etc...
❌ Flatten boxes go in "sheet"!
Please ignore the colors around the examples below.

Examples


Trays and containers

Trays of all shapes, material and colors. Can be rectangular or round, usually used for food.
❌ Don't mistake with box! See examples below...
Please ignore the colors around the examples below.

Examples


Cup shaped / cylinder

Any cup shaped or cylindric object. Includes plastic cups, fast food cups, coffee cups, wooden cylindric boxes, etc...
❌ If the object is obviously a food tray, please use "trays and containers" which is more specific.

Examples


Lid / cap / disk

Any circular flat object. Includes cup lids, soda/beer caps (caspules) or any flat disk.
Please ignore the colors around the examples below.

Other / unsure

If none of the categories fit, use this "other" category. If you're not sure, please also use this category.


Colors

White-ish

White or particularly light color. If most of the object is white, please use this category.

Examples


Black-ish

Black or particularly dark color. If most of the object is black, please use this category.

Examples


Colored

Any color(s) that are not black or white, multicolor.
❌ If most of the object is white or black, please use the corresponding category.

Examples


Translucid

Completely transparent / see through matter (like glass, transparent plastics), where a color is not identifiable.

Examples


Opacity

Transparent / low opacity

The object is see-through, sheer, revealing what is behind it. Even if the content behind cannot be clearly identified, it is still considered transparent if its presence can be guessed.
✅ Slighly see-though is still in this category.

Examples


Opaque

Opaque, no transparency, no way of seeing anything through.

Examples


Design

Pattern / design

The object has a design, a logo, a textured pattern, etc...

Examples


Text / bar code

The object shows some text or a bar code.

Examples


Solid (plain) color

The object has only one color.

Examples


Material

Material

You will have to choose the material from the following list:

  • Wood
  • Glass
  • Plastic
  • Metal
  • Paper / cardboard
  • Ceramic / clay
  • Fabric
If you can't find a fitting material or you're hesitating on the category, please use the "Other material / unsure" category.
